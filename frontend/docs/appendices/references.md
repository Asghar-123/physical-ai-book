# References

This section provides a list of key resources, official documentation, and academic papers that informed the content of this textbook. It is crucial to consult these primary sources for the most up-to-date and in-depth information.

## General Robotics & AI

-   **ROS 2 Documentation**: The official documentation for the Robot Operating System 2.
    -   [https://docs.ros.org/en/humble/index.html](https://docs.ros.org/en/humble/index.html) (for Humble Hawksbill)
    -   [https://docs.ros.org/en/iron/index.html](https://docs.ros.org/en/iron/index.html) (for Iron Irwini)
-   **Gazebo Documentation**: The official documentation for the Gazebo simulator.
    -   [http://gazebosim.org/tutorials](http://gazebosim.org/tutorials)
-   **NVIDIA Isaac Sim Documentation**: Official resources for NVIDIA Isaac Sim.
    -   [https://docs.omniverse.nvidia.com/app_isaacsim/app_isaacsim/overview.html](https://docs.omniverse.nvidia.com/app_isaacsim/app_isaacsim/overview.html)
-   **NVIDIA Isaac ROS Documentation**: Official documentation for the Isaac ROS modules.
    -   [https://nvidia-isaac-ros.github.io/](https://nvidia-isaac-ros.github.io/)
-   **OpenAI Documentation**: Official documentation for OpenAI APIs, including Whisper and GPT models.
    -   [https://platform.openai.com/docs/](https://platform.openai.com/docs/)

## Specific Technologies

-   **FastAPI Documentation**: Modern, fast (high-performance) web framework for building APIs with Python 3.7+ based on standard Python type hints.
    -   [https://fastapi.tiangolo.com/](https://fastapi.tiangolo.com/)
-   **Qdrant Documentation**: Vector similarity search engine.
    -   [https://qdrant.tech/documentation/](https://qdrant.tech/documentation/)
-   **Neon Documentation**: Serverless Postgres.
    -   [https://neon.tech/docs/](https://neon.tech/docs/)
-   **Docusaurus Documentation**: Build optimized websites quickly, focus on your content.
    -   [https://docusaurus.io/docs](https://docusaurus.io/docs)
-   **MoveIt Documentation**: The most widely used software for mobile manipulation and provides an easy-to-use platform for developing new robotic applications.
    -   [https://moveit.ros.org/documentation/](https://moveit.ros.org/documentation/)
-   **Nav2 Documentation**: The ROS 2 Navigation Stack.
    -   [https://navigation.ros.org/](https://navigation.ros.org/)

## Academic Papers & Research (Examples)

-   **ROS 2: Design, Architecture and Future**: M. D. F. P. A. Gerkey et al., 2017.
    -   *Note*: This is an older paper, but fundamental for understanding the transition from ROS 1 to ROS 2.
-   **Humanoid Robot Locomotion: A Review**: Provides an overview of various locomotion strategies and challenges for humanoid robots.
-   **Deep Learning for Robot Perception: A Review**: Discusses the application of deep learning in robot vision, object detection, and segmentation.

This list is not exhaustive but provides a strong foundation for further exploration into the topics covered in this book.
